# Ліки Поради

Android-додаток з інтерактивним силуетом людини для пошуку ліків за симптомами.

## Збірка APK через GitHub Actions

1. Завантажте цей репозиторій на GitHub
2. Перейдіть до **Actions** → **Build APK** → **Run workflow**
3. Після завершення скачайте APK з розділу **Artifacts**

## Структура

- `app/src/main/assets/index.html` — весь застосунок (HTML + CSS + JS)
- `app/src/main/java/.../MainActivity.java` — WebView обгортка
